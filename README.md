## task 4
```
Задача решена без вспомогательных функций, использования дополнительных массивов, посредством функции _arrayOffset_, которая принимает на вход параметры типа int - массив чисел и флаг смещения массива. В функции реализована следующая логика:
1. Принятый на вход флаг приводится к числу, соответствующий размерности принятого на вход массиву;
2. Переменная _firstCompletion_ выявляет количество блоков перебора данных, в соответствии с целым остатком, приведенного к истинности флагом, от размерности массива.
3. Блок while осуществляет данную перестановку;
4. Если остаток размерности массива от приведенного флага не равно нулю, то очередной цикл осуществляет перестановку данных до того момента, пока не дойдет до индекса начальной замены;
5. Если остаток размерности приведенного флага от ☺статка длины массива к приведенному флагу не равно нулю, то, на найденную разность, осуществляет доперестановку данных.
6. Вывод в терминал результатов.
```
